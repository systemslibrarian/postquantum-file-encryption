PostQuantum.FileEncryption
Browser demo — encrypt & decrypt files, fully client-side
GitHub ↗

Pick a file, choose a passphrase, and encrypt it into a .pqfe container — then decrypt it back. Everything runs in your browser via WebAssembly; your file never leaves this device.

What's under the hood: a Rust → WebAssembly core that re-implements the documented .pqfe format — passphrase-based AES-256-GCM with PBKDF2-HMAC-SHA256. It is byte-compatible with the .NET library (verified against the same known-answer test vectors), so a file encrypted here decrypts with the library, and vice versa.
