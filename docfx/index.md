# PostQuantum.FileEncryption — API Reference

A high-level, fail-closed file and stream encryptor for .NET. The stable core is symmetric and
passphrase-based (AES-256-GCM with PBKDF2 or Argon2id); ML-KEM-768 recipient mode is experimental.

- **[API documentation](api/index.md)** — generated from the library's XML docs.
- Project README, format spec, threat model, and roadmap live in the
  [GitHub repository](https://github.com/systemslibrarian/postquantum-file-encryption).

Start with `PqFileEncryptor` and `PqFileDecryptor`.
